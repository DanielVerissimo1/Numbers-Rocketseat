**Numbers** é um desafio do curso de **JavaScript - Rocketseat**, nesse desafio é necessário desenvolver um site simples para geração de números aleatórios dentro de um intervalo definido pelo usuário, com a opção de evitar repetições e com feedback visual animado dos resultados.

## 🚀 Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| 🎲 **Sorteio personalizado** | Escolha a quantidade de números a serem sorteados |
| 📊 **Definição de intervalo** | Defina o valor mínimo e máximo do sorteio |
| 🔁 **Sem repetição** | Opção para evitar números repetidos no sorteio |
| 📱 **Responsivo** | Interface adaptada para desktop e mobile |
| ✅ **Validações** | Verifica se o intervalo e a quantidade são válidos |
| 🎨 **Feedback visual** | Animações e notificações para ações do usuário |
